## 🛠️ Development Guide

### Building and Running
Navigate to the `my-profile/` directory to execute commands:

- **Dev Server**: `npm run dev` (Starts at http://localhost:3000)
- **Build**: `npm run build`
- **Lint**: `npm run lint`
- **Start**: `npm run start`
